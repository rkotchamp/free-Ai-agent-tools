# Contributing

Thanks for helping improve this list.

## Rules for new tools

- Link only to the official website or official GitHub repository.
- Do not repeat links already in the README.
- Include a short description of what the tool does.
- Mention the free/open-source status clearly.
- Prefer tools that are actually useful for AI agents, multi-agent workflows, RAG, coding agents, or automation.

## Weekly update checklist

- [ ] Check that every new URL is official.
- [ ] Check that the URL is not already listed.
- [ ] Confirm the tool still has a free tier, free self-host option, or open-source version.
- [ ] Keep descriptions short and neutral.
